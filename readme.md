Шо це таке? Колхозная библиотека для практики в 3D-рендеринге. Основана на материале из [этой](https://habrahabr.ru/post/248153/) статьи на “Хабре”, плюс, собственных модификациях

  
\#0 commit - от исходного кода из 2-ой статьи остались парсер для моделей и класс TGAImage. Остальное - либо заметно переработано, либо заменено (как geometry.h на библиотеку GLM), либо вырезано

  
Что создано?

* Отрисовка отрезков при помощи алгоритма Брезенхема  
* Отрисовка каркаса  
* Отрисовка полигонов с примитивным освещением

  
\#1 commit - добавление библиотек GLFW и GLM в Git, обновление readme.md

  
\#2 commit - Патчи и правки (исправление ранее не замеченной ошибки компиляции, возможность чтения нескольких моделей в разных путях, подключение Vulkan API)

  
\#3 commit - Z-буфер

  
\#4 commit - Текстурирование
